Instructions

It is common to visualize data that varies across different countries on a map, such that regions with a higher value are assigned a darker color.

Assuming that we are using 5 colors, there are various methods to group the data into 5 groups for the color assignment.

  • [W = 0] Each color occupies a roughly equal area on the screen.
  • [W = 1] Each color is used for the same number of countries. (For example, if we have 200 countries, each color will be used for 40.)
  • [0 < W < 1] A mixture between the two above. (Trying to strike a balance between the two methods.)

Move the blue interactive slider above the maps to see how changing the value of W affects the result.

Note that regardless of the value of W, the map is presenting the same information, but the coloring (ranges/legend) is done differently.
